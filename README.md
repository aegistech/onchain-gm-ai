# FunDAO Protocol

Decentralized crowdfunding platform on Base.

## Deployment Instructions

1. **GitHub**: Push this code to a GitHub repository.
2. **Vercel**: Import the project from GitHub.
3. **Environment Variables**:
   In Vercel Project Settings > Environment Variables, add:
   - Name: `API_KEY`
   - Value: `YOUR_GEMINI_API_KEY_HERE` 
   
   *(Use the key you have: AIzaSy...)*

4. **Deploy**: Click Deploy. The build command is pre-configured to `vite build`.
